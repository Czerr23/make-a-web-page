Lesson One: The Basics of the Web

An Overview of the Web

The web consist of a vast amount of computers from all over the world that interconnect with each other. When someone goes to a web page, the information entered is a HTTP Request.

This request is uses a HyperText Transfer Protocol(HTTP), which is sent to a server. The server replies finding the correct HTML document which the web browser interprets and displays it on the computer screen. Watch this video to have a better concept on the information I just presented.

Major Pieces of the Web
  • HTTP-The main protocol of the web
  • Servers-Computers that host the files that make uo the web
  • Internet-The world's largest computer network
  • Browser-A program that runs on your computer to display files found on the web

This Video will explain the process.

HTML

HTML is an acronym for HyperText Markup Language and in essence is heart of the web. HTML is composed of:

  • Text content "what you see"
  • Markup "how it looks"
  • References to other documents eg. images and videos
  • Links to other pages

To explain in full here is a video.

HTML Markups

HTML markups are made up of tags. They start with a opening tag and end with a closing tag. Inbetween the opening and closing tags is the content, this is also a tag. These tags all together as a whole arecalled an element.

Some Example of Tags
  • b = Bold
  • em = Italic
  • p = Paragraph
  • br = Line break
  • img = Images
Elements (Inline and Block)

There are two types of elements inline and block. Inline only occupies the space along the content. Block forms a box or block that works like a container around the content.

Lesson Two: Making a Scructured Document with HTML

Developer Tools

These tools exist online to assist in every aspect of developing.

The "Tree-Like Structure"

HTML Elements can have other elements inside of them that link to other elements which causes a "tree-like" structure. This structure looks like a grouping of boxes within each other.

The relationship between indented HTML and Boxes

The indentented HTML code is located on the box which the content is associated with.

Text Editors

To write code more efficiently, programmers will use text editors. Some of these editors will help by creating a closing tag automatically when you first write an opening tag.

Lesson 3: Adding CSS Style to HTML Structure

Avoiding Repetition
  • Saves a programmer time
  • Helps avoid errors
  • Helps maintain consistency throughout the code

CSS

CCS stands for Cascading Style Sheets. It allows for control over the style of HTML elements. With it, you can assign a class to HTML elements. This change allows the style of all elements to belong to that class.
